Pytris

Pytris is a simple Tetris clone built with Python and Pygame, a python libary designed to make games. I developed Pytris for a university CS class final project to showcase my ability to design, implement, and demonstrate a Python program of a larger scale than the course assignments I had done so far. I choose Tetris because I wanted to do something simple but outside the scope of the material I learned in the class. If u want to take a look at the source, I share the code on github.

Demo

This is a short demo of the user experience of Pytris showcasing a user playing a single game and getting game over.

Road Rage

Road rage is a simple 2D car dodging mini-game I added into the Pytris GUI to expand upon the project and increase the undertaking.
