Late Checkout Leaderboard

LC Leaderboard is a Web3 leaderboard built for product studio Late Checkout. Users can add new peers, send $LC token, and are ranked against each other based on $LC holding. The code is available on github.

Trade $LC Tokens

Users login using Rainbowkit wallet and add peers to the database where name and address are stored. The app then pulls all the user's token holdings ranking them. The app also allows users to send token between team members.
