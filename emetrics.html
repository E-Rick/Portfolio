Emetrics

Emetrics is a email campaign analytics web app that allows users to send out email campaigns or surveys and tracks everything from open rates to responses. Emetrics uses a credit based system powered by Stripe payments for users to send out campaigns. I am using the MERN (MongoDB, Express, React, Node) tech stack to build Emetrics. The web app is currently under development and you can view the source and a more technical description on github.
